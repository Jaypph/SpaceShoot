# 🚀 SpaceShoot
**An intense, web-based Tower Defense experience.**


SpaceShoot is a fast-paced tower defense game built with modern web technologies. Defend your base against waves of enemy spacecraft by placing strategic turrets and managing your resources.

## ✨ Features
- **Classic TD Gameplay:** Strategic turret placement and resource management.
- **HTML5 Canvas:** Smooth graphics and animations powered by CreateJS.
- **Scaling Difficulty:** Enemies get stronger as you progress through waves.
- **Open Source:** Originally based on the "Turrets" project by [msakuta](https://github.com/msakuta), now expanded with new features.

## 🛠️ Built With
SpaceShoot is crafted using:
*   **HTML5 & CSS3** - Core structure and styling.
*   **JavaScript** - Game logic and engine.
*   **CreateJS / EaselJS** - Sprite handling, animations, and rendering.
